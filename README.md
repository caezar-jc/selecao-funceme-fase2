Os vértices podem ser de 3 tipos diferentes (Fornecedor, Junção ou Cliente).</h3>
<p>Quando o vértice for:</p>
<ol>
<li>Fornecedor, o valor de ponto flutuante associado é a quantidade de produto que ele pode entregar a um cliente.</li>
<li>Junção, ele deve repassar igualmente o valor recebido pelo fornecer para os clientes ligados a ele.</li>
<li>Cliente, o valor de ponto flutuante associado é a quantidade de produto que ele deve pedir a um fornecedor ligado a ele. O cliente sempre possuirá a mesma demanda.</li>
<li>Um Fornecedor pode estar ligado (--&gt;--) somente a um cliente ou uma junção.</li>
<li>Uma junção sempre deve estar ligada a um fornecedor e a um cliente.</li>
<li>Uma junção não pode estar ligada a uma junção</li>
<li>Somente existirá um único fornecedor</li>
<li>Poderá existir uma ou mais junções</li>
<li>Poderá existir um ou mais clientes</li>
</ol>
